    Lessions

    After each lession, redeploy your container and watch in your webbrowser and deploymentconsole.
    1) ENV-Variables
    1. Change ENV-Variable LOGLEVEL_APP
    2) Secrets
    1. Create a secret with a File logging.properties
    2. Define LOGLEVEL_APP in this file
    3. Mount this secret in your Container on /secrets/
    3) Configmap
    1. Create a Configmap with a file logging.properties
    2. Define LOGLEVEL_DOMAIN in this file
