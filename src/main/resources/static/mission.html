Mission

A) Environment
  1. Setzen sie die ENVIRONMENT-Variable LOG_LEVEL auf:
    • INFO
    • DEBUG
    • FATAL
    • TRACE
    • OFF
    • ERROR
  2. Optional kann der gesamte RootLogger mit ROOT_LOG_LEVEL gesetzt werden.
B) Secrets
  1. Erstellen sie zwei Secrets namens:
    • user.secret
    • password.secret
    und schreiben sie einen wahlfreien Inhalt hinein.
  2. Verwenden sie die Secrets auf dem Pfad /secrets/
